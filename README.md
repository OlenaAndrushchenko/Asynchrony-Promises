## Practice

```json

    "results": [
        {
            "title": "Learn Js",
            "priority": "alta",
            "isDone": false
        },
        {
            "title": "Estudiar CSS",
            "priority": "alta",
            "isDone": true
        },
        {
            "title": "Estudiar OOP",
            "priority": "media",
            "isDone": false
        },
        {
            "title": "Study AI",
            "priority": "baja",
            "isDone": false
        }
    ]
}
```

2. Use the same JSON to print a table with all the 'All' elements and their properties in the browser document.
